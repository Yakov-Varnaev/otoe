# otoe
This is simple tool for espanso users. It allows to use obsidian as note editor and then with just one command process them to match config.


structure for md files:

```
obsidian_root/
  project_name/
    file1.md
    file2.md
    file3.md
  other_project/
    file1.md
    file2.md
    file3.md
```
